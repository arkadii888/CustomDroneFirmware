# About the Project

This project involves developing drone control software from scratch. At its core is a drive for a deep understanding of all processes occurring within the flight controller during operation.

## Development Goals
**Learning the Fundamentals:** Mastering flight physics and mathematical control models.

**Peripheral Interaction:** Deep-diving into sensor operations (accelerometer, gyroscope, barometer, power monitoring) over I2C and SPI buses.

**Control Algorithms:** Independent implementation of stabilization and signal processing algorithms.

**Control & Transparency:** All code is written manually without AI to ensure complete comprehension of every single line of the program.

## Tech Stack
**Platform:** ESP32-S3 (M5Stack Stamp Fly)

**Framework:** ESP-IDF (v5.2+)

**Language:** C++
